# Vision EVM

# What is it?
An experimental Ethereum Virtual Machine written in C for learning purposes.

# Dependencies
Clang && LLVM

# Building
```
make
```

# Running
```
./bin/main
```

# Testing
#### All tests reside in a single program and can be run compiled by running:
```
make tests
```
#### Running tests:
```
./tests/bin/tests/main_t
```
